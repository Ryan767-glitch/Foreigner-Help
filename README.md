# 日本生活SOS

外国人・留学生・技能実習 / 特定技能・旅行者向けに、病院、避難所、災害情報、役所手続き、相談窓口、生活フレーズを1か所にまとめた静的サイトです。

## コンセプト
- 緊急時に最短で必要な情報へ到達できる「SOSファースト」設計
- 複雑で読みにくい公的情報を、民間品質のUI/UXで再構成
- BtoC（広告、保険、通信、送迎、翻訳提携）と BtoB（学校、大学、企業、監理団体、登録支援機関）両方の展開を想定

## 含まれる内容
- ヒーローセクションと価値提案
- クイックアクション導線
- ユーザータイプ別の情報整理
- 緊急時の番号・医療フレーズ
- 災害情報ハブ
- 生活立ち上げガイド
- 公的相談窓口リンク
- B2B導入提案セクション

## localhost で確認する
最短では次のコマンドで起動できます。

```bash
npm start
```

起動後、ブラウザで `http://localhost:4173` を開いてください。

## 代替コマンド
Node のスクリプトを使わずに直接起動する場合は以下でも確認できます。

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

## デプロイ
静的サイトのため、Vercel などのホスティングにそのまま配置できます。
