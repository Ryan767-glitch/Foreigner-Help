# 東京23区 外国人向け行政手続きナビ

日本に来た外国人が、東京都23区で生活を始めるために必要な主な手続きを確認できるシングルページWebアプリです。

## 主な機能
- 23区の選択と公式サイトへの直接アクセス
- 在留資格・入国状況・年齢・家族有無に応じたチェックリスト生成
- 印刷 / PDF 保存
- 行政ポータル風の落ち着いたUI（優先度表示付き）

## 使い方
`index.html` をブラウザで開くだけで利用できます。

ローカルサーバーで確認する場合:

```bash
python3 -m http.server 4173
```

## 注意
本アプリは一般的な案内です。申請要件は変更されることがあるため、最終的には各区・出入国在留管理庁等の公式情報を必ず確認してください。

## Vercel へのデプロイ
このリポジトリは静的サイトなので、そのまま Vercel にデプロイできます。

1. Vercel CLI をインストール（ローカル環境で実行）
   ```bash
   npm i -g vercel
   ```
2. プロジェクト直下でログイン・初回デプロイ
   ```bash
   vercel
   ```
3. 本番反映
   ```bash
   vercel --prod
   ```

`vercel.json` を追加済みのため、クリーンURLと基本セキュリティヘッダーが適用されます。
